# Manual Test Checklist

- [ ] Launch game and confirm build number is visible at bottom-left.
- [ ] Open crafting menu and craft at least two different weapon recipes.
- [ ] Open ship editor and verify per-item crafted modules appear in the palette.
- [ ] Drag crafted modules into ship layout and save.
- [ ] Fire equipped crafted weapons and confirm damage/firing still works.
- [ ] Upgrade a crafted module in the palette context menu and verify upgrade feedback.
- [ ] Recycle a crafted module and confirm partial resource refund appears.
